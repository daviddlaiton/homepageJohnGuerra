Navigation widget for large collection of files
Widget for easily navigation through large collection of files or photos.

Undergrad project by Andrés David Laiton Vargas working under the supervision of John Alexis Guerra Gómez





Github Demo Poster Thesis

Objective

Create a widget that allows easily navigation through large collection of files or photos.

Problem Statement

Traditional tools for exploring files do not provide a usable easy-to-use interface for users to interact with. Additionally, all the existing solutions follow similar ways to show the files to the user. These solutions only show few information about the files, despite of the different way to show it. As a solution of the problems mentioned before we present the navigation widget. This widget is made to provide an easy-to-use tool for the user that shows all the information available. This widget is implemented in Photoring (an alternative tool for the navigation, exploration and visualization of large photo collection) as an example of how it works. The navigation widget can be used in other tools for exploring files or photos, how is it done in Photoring.

Deliverables

  • Nagivation widget implemented in file explorer.
  • Custom navigation widget.
