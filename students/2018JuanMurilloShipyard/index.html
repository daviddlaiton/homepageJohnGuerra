Shipyard
A web application to ease the exploration, navigation and summarization of large multidimensional datasets

Undergraduate project by Juan Guillermo Murillo Castillo working under the supervision of John Alexis Guerra Gómez





Objective

Develop a web application that allows any user to use Navio with multi-dimensional datasets of different file extensions without programming. Also, the user will be able to filter, explore, chain queries, summarize and navigate the dataset. Finally, he or she could export the filtered data or just the visualization.

Problem Statement

Have you ever wondered how to explore, navigate and summarize large datasets in a faster and easier way? Data scientists could use different approaches to accomplish that like using programming languages to create a visualization or load the dataset into a visualization tool like Tableau, initially presented as Polaris, or Voyager. Those approaches require time, basic knowledge about programming or visualization techniques. The probxwºlem is that the tasks of explore, navigate and summarize large multidimensional datasets with a programming language or a visualization tool require visualization design expertise, time and knowledge in programming. To deal with this we present Shipyard: an open source web application that eases the tasks stated above.

Deliverables

An usable web application that allow users:

  • Drag and drop different file types (CSV, TSV, TXT and with custom separators)
  • Navigate, explore and summarize data using Navio
  • Execute chainable queries while previewing the filtered data
  • Export data filtered
  • Export visualization (Navio embedded and set up)
